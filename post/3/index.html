链表学习笔记----单链表

发表于 2022-07-30 分类于 技术

一、单链表

单链表中的每个节点不仅包含值，还包含链接到下一个节点的引用字段。通过这种方式，单链表将所有节点按顺序组织起来。

节点结构

// Definition for singly-linked list.
typedef struct List
{
    int data;
    struct List *next;
}List,*pList;

操作

  1. 访问操作

    与数组不同，我们无法在常量时间内访问单链表中的随机元素。如果我们想要获得第$i$个元素，我们必须从头节点逐个遍历。我们按索引来访问元素平均要花费$O(N)$时间，其中N为链表长度。

  2. 添加操作

    如果我们想在给定的节点$prev$之后添加新值：

    ① 使用给定值初始化新节点$cur$。

    ② 将$cur$的$next$字段链接到$prev$的下一个节点$next$。

    ③ 将$prev$中的$next$字段链接到$cur$。

    $与数组不同$，我们不需要将所有元素移动到插入元素之后。因此可以在$O(1)$时间复杂度中将新节点插入到链表中。

  3. 删除操作

    从单链表中删除现有节点$cur$：

    ①找到$cur$的上一个节点$prev$及其下一个节点$next$。

    ②接下来链接$prev$到$cur$的下一个节点$next$。

    第一步中，需要找出$prev$和$next$。使用$cur$的引用字段很容易找出$next$，但是我们必须从头节点遍历链表，以找出$prev$，它的平均时间是$O(N)$,其中N是链表的长度。因此，删除节点的时间复杂度是$O(N)$。空间复杂度为$O(1)$，只需常量空间来存储指针。

设计链表

单链表中的节点应该具有两个属性：$val$ 和 $next$。$val$ 是当前节点的值，$next$ 是指向下一个节点的指针/引用。如果要使用双向链表，则还需要一个属性 $prev$ 以指示链表中的上一个节点。假设链表中的所有节点都是 $0-index$ 的。

在链表类中实现这些功能：

$get(index)$：获取链表中第 $index$ 个节点的值。如果索引无效，则返回$-1$。
$addAtHead(val)$：在链表的第一个元素之前添加一个值为 $val$ 的节点。插入后，新节点将成为链表的第一个节点。
$addAtTail(val)$：将值为 $val$ 的节点追加到链表的最后一个元素。
$addAtIndex(index,val)$：在链表中的第 $index$ 个节点之前添加值为 $val$ 的节点。如果 $index$ 等于链表的长度，则该节点将附加到链表的末尾。如果 $index$ 大于链表长度，则不会插入节点。如果$index$小于$0$，则在头部插入节点。
$deleteAtIndex(index)$：如果索引 $index$ 有效，则删除链表中的第 $index$ 个节点。

定义链表结构体
//定义链表结构体
typedef struct MyLinkedList_t
{
	int val;//值
	struct MyLinkedList_t *next;//指向下一个节点的指针
}MyLinkedList;

注意：如果在这个struct定义里，直接把next指针定义为MyLinkedList *next就会报错，因为别名MyLinkedList是在结构体定义结束后才可以识别的一个别名；

创建链表
//创建链表
MyLinkedList* myLinkedListCreate()
{
    //使用虚拟头节点作为访问、管理链表的入口
    MyLinkedList *obj = (MyLinkedList *)malloc(sizeof(MyLinkedList));
    //对虚拟链表头进行初始化，此时真实的链表为空
    obj->val = 0;
    obj->next = NULL;
    //返回链表入口
    return obj;
}
在链表头部增加节点
//在链表头部增加节点
void myLinkedListAddAtHead(MyLinkedList *obj,int val)
{
    MyLinkedList *node = (MyLinkedList *)malloc(sizeof(MyLinkedList));
    node->val = val;
    //将新节点插入至虚拟头节点和原来虚拟头节点下一个节点之间的位置，亦考虑了真实链表为空的情况
    node->next = obj->next;
    obj->next = node;
}
在链表尾部增加节点
//在链表尾部增加节点
void myLinkedListAddAtTail(MyLinkedList *obj,int val)
{
    MyLinkedList *node = (MyLinkedList *)malloc(sizeof(MyLinkedList));
    //由于真实链表可能为空，所以标记位置从obj开始
    MyLinkedList *nowList = obj;
    node->val = val;
    node->next = NULL;
    //一直遍历，直到当前节点的下一个节点为空，当前节点即为尾部节点
    while(nowList->next != NULL)
    {
        nowList = nowList->next;
	}
    //将原来的尾部节点的下一个节点指向新节点
    nowList->next = node;
}
获取链表中第 $index$ 个节点的值，从0开始
int myLinkedListGet(MyLinkedList *obj,int index)
{
    MyLinkedList *nowList = obj->next;
    int cnt = 0;
	//处理异常情况，index小于0，真实链表为空
    if(index < 0 || nowList == NULL)
    {
        return -1;
	}
    //进行链表遍历，循环终止条件：cnt位置达到index位置
    //break条件:当前位置节点为空，返回-1
    while(cnt < index)
    {
        if(nowList == NULL)
        {
            return -1;
		}
        nowList = nowList->next;
        cnt++;
	}
    //循环结束后，取到了当前位置为index的节点，需判断该点为不为空
    if(nowList != NULL)
    {
        return nowList->val;
	}
    return -1;
}
在链表中的第 $index$ 个节点之前添加值为 $val$ 的节点
void myLinkedListAddAtIndex(MyLinkedList *obj,int index,int val)
{
    MyLinkedList *nowList = obj;
    int cnt = 0;
	//如果为负索引或者0，直接头插
	if(index <= 0)
	{
		myLinkedListAddAtHead(obj,val);
        return;
	}
    //遍历链表，break条件：cnt位置到达index位置,此时nowList节点位置到达index-1位置
	while(nowList->next != NULL)
	{
		if(cnt == index) {break;}
		nowList = nowList->next;
		cnt++;
	}
    //如果不满足该条件，一定是通过nowList->next == NULL跳出循环的，index无效
	if(cnt != index)
	{
		return;
	}
    //初始化新节点，将新结点的next节点指向index节点，将index-1位置节点的next节点指向新节点
	MyLinkedList *node = (MyLinkedList *)malloc(sizeof(MyLinkedList));
	node->val = val;
	node->next = nowList->next;
	nowList->next = node;
}
删除链表中的第 $index$ 个节点
void myLinkedListDeleteAtIndex(MyLinkedList *obj,int index)
{
    MyLinkedList *nowList = obj->next;
    int cnt = 0;
	//处理异常情况 负索引或者真实链表为空，直接退出
    if(index < 0 || obj->next == NULL)
    {
        return;
	}
    //此时真实链表至少有一个节点，所以可以直接删除第一个
    if(index == 0)
    {
        obj->next = obj->next->next;
        return;
	}
    //遍历链表，循环终止条件：当前节点为最后一个节点；break条件:当前节点为index位置的前一个节点
    while(nowList->next != NULL)
    {
        if(cnt == index - 1) {break;}
        nowList = nowList->next;
        cnt++;
	}
    //需满足index位置有节点，且当前节点为index-1位置上的节点，才能进行删除操作
    if(cnt == index - 1 && nowList->next != NULL)
    {
        nowList->next = nowList->next->next;
	}
}
实现free接口，释放已经申请的内存

一个链表，每一个节点都需要释放。但是如果我们直接从第一个节点释放，那么后面的next就无法取到了，链断掉以后，剩下的内存就再也无法访问。所以链表释放内存必须从最后一个开始释放，在释放倒数第二个…依次释放。但是我们遍历链表，从最后一个节点释放，但是链表是单向的，我们无法通过链表取到它上一个节点。所以需要递归。

每次只要检查到当前节点还有下一个节点，就先去释放下一个节点，直至找到最后一个节点，然后逐层往回走，最后释放整个链表。

//递归函数
void myNodeFree(MyLinkedList *node)
{
    if(node->next != NULL)
    {
        myNodeFree(node->next);
        node->next = NULL;
	}
    free(node);
}
void myLinkedListFree(MyLinkedList *obj)
{
    myNodeFree(obj);
}
