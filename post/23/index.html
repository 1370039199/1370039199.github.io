循环队列设计

循环队列，可以通过一个数组进行模拟，通过操作数组的索引构建一个虚拟的首尾相连的环。在循环队列结构中，设置一个队尾$rear$与队首$front$，且大小固定，结构如下图所示：

在循环队列中，当队列为空，可知$front = rear$;而当所有队列空间全占满时，也有$front = rear$；为了区别这两种情况，假设队列使用的数组有$capacity$个存储空间，则此时规定循环队列最多只能有$capacity-1$个队列元素.当循环队列中只剩下一个空存储单元，则表示队列已满。

根据以上条件，队列判空的条件是：
$$
front = rear
$$
队列判满的条件是：
$$
front = (rear + 1) \quad mod \quad capacity
$$
对于一个固定大小的数组，只要知道队尾$rear$与队首$front$，即可计算出队列当前的长度：
$$
(rear-front+capacity) \quad mod \quad capacity
$$
循环队列的属性如下：

  • $elements$：一个固定大小的数组，用于保存循环队列的元素。
  • $capacity$：循环队列的容量，即队列中最多可以容纳的元素数量。
  • $front$：队列首元素对应的数组索引。
  • $rear$：队列尾元素对应的索引的下一个索引*。

循环队列的接口方法如下：

  • $MyCircularQueueCreate(int\ k)$：初始化队列，同时$base$数组的空间初始化大小为$k+1$。$front，rear$全部初始化为0。
  • $enQueue(int\ value)$：在队列的尾部插入一个元素，并同时将队尾的索引$rear$更新为$(rear+1)\ mod\ capacity$。
  • $deQueue()$：从队首取出一个元素，并同时将队首的索引front更新为$(front+1)\ mod\ capacity$。
  • $Front()$：返回队首的元素，需要检测队列是否为空。
  • $Rear()$：返回队尾的元素，需要检测队列是否为空。
  • $isEmpty()$：检测队列是否为空，根据之前的定义只需判断$rear$是否等于$front$。
  • $isFull()$：检测队列是否已满，根据之前的定义只需判断$front$是否等于$(rear+1)\ mod\ capacity$。

源码实现：

//定义循环队列结构体
typedef struct
{
    int front;
    int rear;
    int capacity;
    int *elements;
}MyCircularQueue;

//循环队列创建、初始化
MyCircularQueue* myCircularQueueCreate(int k)
{
    MyCircularQueue *obj = (MyCircularQueue*)malloc(sizeof(MyCircularQueue));
    obj->capacity = k + 1;
    obj->rear = obj->front = 0;
    obj->elements = (int*)malloc(sizeof(int) * obj->capacity);
    return obj;
}

//判断队列是否为满
bool myCircularQueueIsFull(MyCircularQueue *obj)
{
    return (obj->rear + 1) % obj->capacity == obj->front;
}

//判断队列是否为空
bool myCircularQueueIsEmpty(MyCircularQueue *obj)
{
    return obj->rear == obj->front;
}

//入队
bool myCircularQueueEnQueue(MyCircularQueue *obj,int value)
{
    if(myCircularQueueIsFull(obj))
    {
        return false;
	}
    obj->elements[obj->rear] = value;
    obj->rear = (obj->rear + 1) % obj->capacity;
    return true;
}

//出队
bool myCircularQueueDeQueue(MyCircularQueue *obj)
{
    if(myCircularQueueIsEmpty(obj))
    {
        return false;
	}
    obj->front = (obj->front + 1) % obj->capacity;
    return true;
}

//返回队首元素
int myCircularQueueFront(MyCircularQueue *obj)
{
    if(myCircularQueueIsEmpty(obj))
    {
        return -1;
	}
    return obj->elements[obj->front];
}

//返回队尾元素
int myCircularQueueRear(MyCircularQueue *obj)
{
    if(myCircularQueueIsEmpty(obj))
    {
        return -1;
	}
    return obj->elements[(obj->rear-1+obj->capacity)%obj->capacity];
}

//释放队列空间
void myCircularQueueFree(MyCircularQueue *obj)
{
    free(obj->elements);
    free(obj);
}

在射频通信程序中，经常使用环形缓冲区作为存储通信中发送和接收的数据，上述代码可变形为通信环形队列作为开发使用。

//定义循环队列结构体
typedef struct
{
    INT16U front;
    INT16U rear;
    INT16U capacity;
    INT8U *elements;
}Queue_t;

//环形队列创建
void Queue_Create(Queue_t *queue,INT8U *buffer,INT16U size)
{
    queue->front = 0;
    queue->rear = 0;
    queue->capacity = size + 1;
    queue->elements = buffer;
}

//判断队列是否为空
bool Queue_IsEmpty(Queue_t *queue)
{
    return (queue->front == queue->rear);
}

//判断队列是否为满
bool Queue_IsFull(Queue_t *queue)
{
    return ((queue->rear+1)%queue->capacity == queue->front);
}

//入队
bool Queue_Push(Queue_t *queue,INT8U val)
{
    if(Queue_IsFull())
    {
        return false;
	}
    queue->rear = val;
    queue->rear = (queue->rear + 1) % queue->capacity;
    return true;
}

//出队
bool Queue_Pop(Queue_t *queue,INT8U *val)
{
    INT8U val;
    if(Queue_IsEmpty())
    {
        return false;
	}
    *val = queue->elements[queue->front];
    queue->front = (queue->front+1) % queue->capacity;
    return true;
}

//获取队列有效数据大小
INT8U Queue_GetValidSize(Queue_t *queue)
{
    return (queue->rear - queue->front + queue->capacity) % queue->capacity;
}

//队列刷新
void Queue_Flush(Queue_t *queue)
{
    queue->front = 0;
    queue->rear = 0;
}
